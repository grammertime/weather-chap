# Wardrobe Photos (Phase 2)

Add your clothing photos here. Then update `wardrobe.json` in the project root:

- Set `"image": "your_filename.jpg"` for items you've photographed
- Example: `"Beanie": { "label": "My gray beanie", "image": "beanie.jpg" }`

Place `beanie.jpg` in this folder (`static/wardrobe/`).
